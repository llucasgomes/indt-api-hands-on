## ⚙️ Backend

O backend foi desenvolvido em:

- **Fastify** – framework web rápido e leve para Node.js
- **TypeScript** – tipagem estática
- **Zod** – validação de dados e schemas

<!-- ### Endpoints Principais -->

- `GET /api/users` – endpoint para buscar todos os usuarios
- `POST /api/users` – endpoint para criar um novo usuario
- `GET /api/users/{id}` – endpoint para buscar UM os usuario
- `PUT /api/users/{id}` – endpoint para editar UM os usuario
- `DEL /api/users` – endpoint para deletar UM os usuario

- `GET /docs` – documentação interativa da API (Scalar/OpenAPI)

A documentação completa da API estará disponível em: `${URL_BASE}/api/docs`
